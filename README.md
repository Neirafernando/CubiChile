# CubiChile

Aplicación de escritorio para cubicaciones, presupuestos e informes de obra en Chile.

## Funciones principales

- Gestión de proyectos
- Cubicaciones por partida
- Presupuesto con precio unitario
- Exportación a Excel
- Exportación a PDF
- Informe técnico completo

## Tecnologías

- Python
- PySide6
- SQLite
- ReportLab
- OpenPyXL

## Ejecución local

```bash
source venv/bin/activate
python3 main.py
